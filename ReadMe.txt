●【2_RollingBall】

　　・3年次に制作した作品です。
    ・AppStoreとGooglePlayに作品を公開したいといった目的で制作しました。
    ・ProjectProposalフォルダ内にあるRollingBall.pdfにインストールQRコードと
　　　URLがございますので、ぜひインストールして頂ければ幸いです。

【制作期間】3ヶ月（AppStore公開まで含む）
【制作人数】1人
【動作環境】Window10
【開発環境】Unity 2019.2.21f1 (64-bit)
【開発言語】C#
【フォルダ構成】・exeフォルダ
　　　　　　　　　実行データが入っているフォルダです。
                  Rolling Ball.exeをダブルクリックで
                  起動できます。
　　　　　　　　 ---------------------------------------------　　
　　　　　　　　・projectフォルダ
　　　　　　　　　作品のプロジェクトフォルダです。
　　　　　　　　　Unityを起動し、リストに追加でprojectフォルダ
　　　　　　　　　を追加することで起動できます。
                 ---------------------------------------------
　　　　　　　　・ProjectProposalフォルダ
　　　　　　　　　作品の企画書が入っています。
                 ---------------------------------------------
　　　　　　　　・Game.MOV
                  作品のプレイ動画です。
　　　　　　　　 --------------------------------------------- 

【作品について】
　・スマートフォンを傾けることで、ボールを操作できます。
　・詳細はProjectProposalフォルダ内のRollingBall.pdfを参照

【操作方法】
　・アプリの場合はスマートフォンの傾け
　・PCの場合は「WASD」もしくは「矢印キー」で操作できます。
