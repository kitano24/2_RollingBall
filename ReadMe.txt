●【2_RollingBall】

　　3年次に制作した作品です。
    AppStoreとGooglePlayに作品を公開したいといった目的で制作しました。
    ProjectProposalフォルダ内にある企画書にインストールQRコードとURLが
    ございますので、ぜひインストール頂ければ幸いです。

【制作期間】3ヶ月（AppStore公開まで含む）
【制作人数】1人
【動作環境】Window10
【開発環境】Unity 2019.2.21f1 (64-bit)
【開発言語】C#
【フォルダ構成】・exeフォルダ

【作品について】
　スマートフォンを傾けることで、ボールを操作できます。
　詳細は企画書を参照

【操作方法】
　アプリの場合はスマートフォンの傾け
　PCの場合は「WASD」もしくは「矢印キー」で操作できます。
